## 建模理由
领域模型描述了问题域中事物及其之间的关系与量化的约束，我们需要进一步验证模型的有效性与完备性，管理这些事物的生命周期成为有效的方法 - 状态建模。它主要解决以下问题：

* 从实例的角度识别业务事件，完善、优化业务过程的细节，细化业务过程与领域模型
* 给出业务过程合理性与完备性验证
* 为程序开发提供业务规范细节
